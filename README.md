## ðŸ“Œ Platform Overview
### Admin

- Access all courses, lectures, and uploaded material.
- Approve or reject instructor/admin signup requests.
- Full platform-level monitoring and management.
- Can like and comment on video lectures.
- Cannot modify instructor course content.

### Instructor
- Create and edit courses with a real-time interactive course editor.
- Upload free (and soon, paid) video lectures.
- Edit uploaded videos using an integrated video editing interface.
- Watch, follow, or subscribe to other instructors and courses.
- Start live classes (powered by ZegoCloud).
- Like and comment on video lectures.

### User / Student

- Watch free video lectures.
- Enroll in free courses and access all course materials.
- Join live classes hosted by instructors.
- Like and comment on video lectures.
